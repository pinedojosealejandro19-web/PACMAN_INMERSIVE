# Immersive Pac-Man

Immersive Pac-Man is an open-source web game inspired by the classic arcade experience, redesigned for projection mapping, immersive environments, and creative public play.

This project is built with **HTML, JavaScript, and CSS**, and includes custom sprites, sound effects, keyboard controls, and a QR-based mobile controller powered by Firebase. It was created as an experimental web game that can run directly in the browser and be shared through a public URL.

## Project Purpose

The goal of Immersive Pac-Man is to make a fun, accessible, and expandable version of Pac-Man that can be used in creative installations, classrooms, projection mapping experiments, events, and public interactive experiences.

This project is meant to stay:

- **Open source**
- **Free to use**
- **Free to study**
- **Free to modify**
- **Free to improve**
- **Free to share**

Immersive Pac-Man is not just a finished game. It is a growing creative project that anyone can learn from, remix, upgrade, and expand.

## Features

- Browser-based Pac-Man game
- Built with HTML, JavaScript, and CSS
- QR mobile controller using Firebase
- Keyboard support
- Custom sprites and visual states
- Sound effects and game audio
- Loading screen for asset preparation
- Projection-mapping-ready visual structure
- Multi-surface design potential
- Works through a public web link
- Designed for immersive and experimental setups

## Why This Project Exists

This game was created to explore how classic arcade games can be transformed into immersive public experiences. Instead of being limited to a normal screen, the game can be adapted for projection mapping, large-scale displays, creative rooms, and interactive installations.

The idea is simple:

> Take something familiar, make it bigger, make it playable, and make it open for everyone to improve.
